# Resources

This directory contains application resources.

## icon.png

The system tray icon. To generate it, run on Windows after installing dependencies:

```powershell
python resources/create_icon.py
```

This creates a 64x64 PNG with a blue background and white "A" letter.

## Customizing the Icon

You can replace `icon.png` with any 64x64 PNG image for a custom tray icon.
